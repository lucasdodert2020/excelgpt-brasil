# Roadmap

## Fase 0 - Fundação

- Criar documentação inicial.
- Definir visão, PRD, arquitetura e regras de IA.
- Definir princípios de produto, segurança, SEO e monetização.

## Fase 1 - Base do App

- Criar app Next.js com TypeScript.
- Configurar Tailwind.
- Configurar estrutura de pastas.
- Preparar padrões de componentes e layout.

## Fase 2 - Autenticação e Usuários

- Integrar Auth.js.
- Criar modelo de usuário.
- Criar página de login.
- Criar sessão e área autenticada.

## Fase 3 - Chat MVP

- Criar interface de chat.
- Implementar seletor de versão do Excel.
- Criar endpoint de chat.
- Integrar OpenAI API.
- Salvar mensagens e conversas.

## Fase 4 - Fórmulas

- Geração de fórmulas.
- Correção de fórmulas.
- Explicação de fórmulas.
- Bloco visual para copiar fórmula.
- Observações de compatibilidade por versão.

## Fase 5 - Histórico e Limites

- Listar conversas.
- Reabrir histórico.
- Aplicar limites do plano gratuito.
- Exibir uso mensal.

## Fase 6 - Monetização

- Criar plano Pro.
- Integrar Mercado Pago.
- Integrar Stripe.
- Criar webhooks.
- Controlar status de assinatura.

## Fase 7 - SEO e Conteúdo Público

- Criar páginas públicas de dúvidas frequentes.
- Publicar guias de fórmulas.
- Criar páginas para funções específicas.
- Usar base de conhecimento para conteúdo indexável.

## Fase 8 - Otimização de IA

- Implementar cache.
- Criar base de conhecimento.
- Classificar perguntas frequentes.
- Reduzir custo por usuário.
- Monitorar qualidade das respostas.

## Fase 9 - Expansão

- Upload de planilhas.
- Templates.
- Add-in para Excel.
- Recursos para equipes.
- Integrações avançadas.
