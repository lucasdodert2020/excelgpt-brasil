# SEO

## Objetivo

Preparar o ExcelGPT Brasil para capturar tráfego orgânico de usuários que pesquisam dúvidas de Excel no Brasil.

## Estratégia

- Criar páginas públicas para dúvidas frequentes.
- Criar guias de funções do Excel em português.
- Criar páginas comparando soluções por versão.
- Transformar respostas revisadas da base de conhecimento em conteúdo indexável.
- Usar linguagem natural próxima das buscas reais dos usuários.

## Tipos de Página

### Páginas de Função

Exemplos:

- Como usar PROCV no Excel.
- Como usar SOMASE no Excel.
- Como usar SEERRO no Excel.

### Páginas de Problema

Exemplos:

- Como corrigir erro #N/D no Excel.
- Como somar valores por data no Excel.
- Como procurar valor em outra aba.

### Páginas por Versão

Exemplos:

- Fórmulas compatíveis com Excel 2013.
- Diferenças entre Excel 365 e Excel 2019.
- Fórmulas no Google Sheets em português.

## Requisitos de Conteúdo

- Resposta objetiva.
- Fórmula pronta.
- Explicação passo a passo.
- Compatibilidade por versão.
- CTA para testar no chat.

## SEO Técnico Futuro

- URLs limpas.
- Metadata por página.
- Sitemap.
- Dados estruturados quando adequado.
- Performance alta na Vercel.
- Conteúdo renderizável para buscadores.

## Relação com IA

A base de conhecimento deve alimentar tanto o produto quanto páginas públicas, reduzindo custo de IA e aumentando aquisição orgânica.
