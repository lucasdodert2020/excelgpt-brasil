# PRD - Product Requirements Document

## Objetivo do MVP

Validar um SaaS de IA especialista em Excel para o público brasileiro, com chat, geração de fórmulas, correção, explicação, histórico, login e planos gratuito/pro.

## Escopo do MVP

### Incluído

- Login de usuário.
- Chat para perguntas sobre Excel.
- Geração de fórmulas.
- Correção de fórmulas existentes.
- Explicação passo a passo de fórmulas.
- Seletor de versão do Excel.
- Histórico de conversas.
- Plano gratuito com limites.
- Plano Pro com limites maiores.

### Fora do MVP

- Upload de planilhas.
- Edição direta de arquivos Excel.
- Add-in para Excel.
- Automação via VBA.
- Integrações corporativas.
- Marketplace de templates.

## Personas

### Analista Operacional

Precisa corrigir fórmulas rapidamente e entender erros sem depender de TI.

### Pequeno Empresário

Usa planilhas para controle financeiro, estoque e vendas, mas não domina fórmulas.

### Estudante

Quer aprender Excel com exemplos claros e aplicáveis.

## Jornada Principal

1. Usuário acessa o produto.
2. Faz login ou usa uma experiência gratuita limitada.
3. Escolhe a versão do Excel.
4. Digita uma pergunta ou fórmula.
5. Recebe resposta prática, fórmula e explicação.
6. Pode salvar ou consultar o histórico.
7. Ao atingir limite, recebe convite para plano Pro.

## Requisitos Funcionais

- O usuário deve conseguir enviar perguntas em linguagem natural.
- O usuário deve selecionar a versão do Excel.
- O sistema deve diferenciar geração, correção e explicação de fórmulas.
- O sistema deve salvar histórico de interações.
- O sistema deve aplicar limites por plano.
- O sistema deve exibir respostas em português do Brasil.

## Requisitos Não Funcionais

- Respostas rápidas para dúvidas recorrentes usando cache.
- Interface responsiva.
- Segurança no armazenamento de usuários e histórico.
- Preparação para SEO e páginas públicas.
- Observabilidade mínima para custo e uso de IA.

## Critérios de Sucesso

- Usuários conseguem obter fórmulas utilizáveis.
- Redução de chamadas repetidas à IA por meio de cache.
- Conversão inicial de usuários gratuitos para Pro.
- Páginas públicas começam a capturar tráfego orgânico.
